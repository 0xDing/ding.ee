言葉越しに、はじめまして。丁立（Boris）です。

ホフスタッターとアドラーと王小波、そのあたりの門前を勝手にうろつく犬みたいなものです。ついでに言えば、ヒューム流の不可知論者でもあります。

ここ数年やってきたことは、われながらだいぶ雑多です。UI の設計を描き、ユーザーリサーチやモーションをやり、フロントエンドや iOS を書き、マイクロサービスの基盤やデータシステムを手で組んできました。ホワイトハットとして脆弱性を掘ったこともあります。AI では Agent のオーケストレーション基盤を書き、基盤 LLM や画像生成モデルをゼロから訓練し、マルチモーダルや SFT まわりのオープンソースにも少し関わってきました。起業、プロダクト、システム、AI、クオンツ。前後して、だいたいそのへんを行き来しています。

China Renaissance では、社内起業としてインターネット型の FA プラットフォーム「逐鹿X」を立ち上げました。Sequoia China では、データサイエンスチームをゼロから組成し、プライマリー市場のクオンツにも独自に取り組みました。その後は SaaS をやり、ソフトウェアとドキュメントのあいだにある、もともとかなり怪しい境界を壊そうとしました。少しローコードの入った、両者の中間みたいなものをつくりたかったんです。結果としては、きれいに「早すぎた先駆者」の側に回りました。

厳密にいえば、職業上の肩書きはあまり安定していません。でも、具体的なものをつくること、そして世界をほんの少しでも動かすことだけは、ずっと変わっていません。深さより広さを選ぶことにも、わりと本気で誇りを持っています。

いま主に関心があるのは、金融クオンツと Agentic RL の二つです。

前者に惹かれるのは、もちろん金を稼ぐためだけではありません。市場がおもしろいのは、それが静かでもなければ、理屈どおりにも動かないからです。金融の語りは、人は合理的で、市場は均衡へ向かい、情報は水が池にしみていくみたいに静かに価格へ織り込まれる、という図を好みがちです。でも私は、そういう整いすぎた世界観に昔からあまり惹かれません。むしろ市場は、複雑性経済学や行動経済学のほうから眺めたい。そこでは市場は、混み合い、騒がしく、フィードバックだらけの複雑適応系として立ち現れます。人は偏りや感情、限られた情報、その時々の置かれた状況を抱えたまま流れ込み、互いに影響し合い、そのついでに価格まで書き換えていく。そう考えると、クオンツはずっとおもしろくなります。予測しているのは、単なる数字の列ではありません。物語がどう広がり、確信がどう形づくられ、偏りがどう積み上がり、そしてそれがどう価格になるのか。その過程そのものです。

後者は、知的エージェントが本当に「働く」とはどういうことか、という問いにつながっています。RAG と function calling で止まっているような、おもちゃめいた agent には、正直あまり興味がありません。気になるのは、長期記憶、世界モデル、自己較正、タスク分解、継続学習といった、もっと面倒で、もっと本質的な問題です。私にとって Agentic RL は、流行り言葉というより、かなり素朴でしつこい執念に近い。知的システムが「しゃべれる」から一歩先の「やれる」へ、さらに「やるたびに少しずつ本物になっていく」ところまで行けるのか。ずっとそこを見ています。

このブログでは、プロダクト、システム、AI、クオンツ、美意識のことを書くつもりです。あわせて、判断、方法、創造みたいな、もう少し息の長い問いも書いていきます。テーマはたぶん、いつもきれいには揃いません。でも関心の芯は、だいたい同じです。複雑なものを理解し、新しいものをつくり、変化のなかで自分を何度も調整し直していくこと。

21歳の誕生日に願ったのは、real world を refactor すること。32歳になって願うのは、もっと多くの bias を引き受けて、愛し、生を味わい、作品をつくり、ランダムな摂動を抱きしめること。
